## Overview

**GH0ST_B0y-v_X EV-LLM Oracle** is a VS Codium/VS Code extension that provides an interface for managing and monitoring the EV-LLM Oracle system coordinating **7 blockchains** created from **Genesis Hash 0**.

## Features

- **Oracle VM Control**: Start, stop, and monitor the Oracle VM process
- **7 Genesis Chains Dashboard**: Visual overview of all blockchain states
- **Memory Monitor**: Track resource usage in real-time
- **Status Bar Integration**: Quick access to Oracle status
- **Output Channel**: Detailed logging of Oracle operations

## Commands

| Command | Description |
|---------|-------------|
| `𝔾ℍ𝟘𝕊𝕋_𝔹𝟘𝕐: Start Oracle VM` | Launch the Oracle VM process |
| `𝔾ℍ𝟘𝕊𝕋_𝔹𝟘𝕐: Stop Oracle VM` | Terminate the Oracle VM |
| `𝔾ℍ𝟘𝕊𝕋_𝔹𝟘𝕐: Check Chain Status` | Display status of all 7 chains |
| `𝔾ℍ𝟘𝕊𝕋_𝔹𝟘𝕐: Open Oracle Dashboard` | Open the visual dashboard |

## Configuration

| Setting | Default | Description |
|---------|---------|-------------|
| `ghostBoyOracle.oracleVmPath` | `/home/nuts/.genesis/scripts/oracle_vm.py` | Path to Oracle VM script |
| `ghostBoyOracle.autoStart` | `false` | Auto-start Oracle on extension activation |

## Installation

```bash
cd ghost-boy-oracle-vscode
npm install
npm run compile
```

Then press `F5` in VS Codium to launch the extension in development mode.

To package for distribution:
```bash
npm install -g @vscode/vsce
vsce package
```
